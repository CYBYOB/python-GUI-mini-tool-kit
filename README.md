# python-GUI-mini-tool-kit
python GUI 百度文库、知网直接下载保存、wifi破解器（内含8200+个密码）、古诗生成器（目前只录入了一部分）</br>

## 零、前言
1、本程序只适用于学习、交流，不可用于其他用途。
2、古诗生成器没有做过多的测试等，还是存在一定问题的，诗句是自己根据 github上的相关项目整理并录入了数据库，想要的可以留言（留邮箱、联系方式等）。
古诗，可是我们大中华的瑰宝呀~</br>
